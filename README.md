# 台灣刮刮樂大師模擬器

用 React + Vite 打造的台灣風格刮刮樂模擬器。選擇不同面額的遊戲、刮開卡片、統計你的花費與獎金，並透過互動式畫面了解刮刮樂的機率體驗。

## 特色

- 三種面額的刮刮樂選擇
- 擬真的刮開互動效果與中獎動畫
- 即時統計花費、獎金與勝率
- 適合機率教育與娛樂用途

## 技術棧

- React 19
- TypeScript
- Vite
- canvas-confetti

## 本機開發

**先決條件：** Node.js

1. 安裝相依套件：
   `pnpm install`
2. 啟動開發伺服器：
   `pnpm run dev`
3. 於瀏覽器開啟 Vite 顯示的本機網址。

## 產出與預覽

- 建置：`pnpm run build`
- 本機預覽：`pnpm run preview`

## 注意事項

此專案僅供學習與娛樂使用，請理性對待博弈遊戲。
